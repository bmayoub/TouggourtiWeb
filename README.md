# Touggourti Web Application

### Installation
`git clone https://github.com/mslaraba/TouggourtiWeb.git`

```
cd TouggourtiWeb
npm install
bower install
grunt clean
grunt all
```

NB : 

 - npm => Node JS https://nodejs.org/en/ 
 - bower => `npm install -g bower` 
 - grunt => `npm install -g grunt-cli`

### Execution

```npm start```
